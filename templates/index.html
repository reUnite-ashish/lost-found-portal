📖 How It Works
📝

1. Report

Submit a lost or found item with details like name, category, location and photo.

🤖

2. Auto Match

Our algorithm automatically finds matching lost & found items based on category and description.

🤝

3. Reunite

Submit a claim with proof of ownership. Admin verifies and connects you with the item.

🎓 The Team
Students Project

Lost & Found Portal (ReUnite)

Developed by BCA 6th Semester (Final Year) students of Maharana Pratap PG College, Gorakhpur. This project is designed to help users report lost items and connect them with found items using a smart matching system.

Department: Bachelor of Computer Applications (BCA)
Batch: Final Year (Semester 6) | Session 2025–26
Project Type: Major Team Project (Academic)
Tech Stack: Python (Flask), MongoDB Atlas, HTML, CSS, JavaScript
